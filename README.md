## 5. Getting Started

### Requirements

For building and running the application you need:

- Node.js v20.4.0

### Installation & Run

```bash
$ git clone https://github.com/uju-in/lime-frontend.git
$ nvm use v20.4.0
$ yarn install
$ yarn dev
```
